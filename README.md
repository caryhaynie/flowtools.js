# flowtools.js -- tools for people who write async tools.

## installing 

### via git ( using npm link )

`$ git clone https//github.com/caryhaynie/flowtools.js.git`

followed by:

`$ cd flowtools.js; sudo npm link`

`$ npm link flow.js`

### via npm

`$ npm install flowtools.js`
